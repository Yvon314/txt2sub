txt2sub crée un fichier sous titre à partir du texte copié sur youtube (copier/coller transcription)


requirement 
installer python

usage:
sous windows:

par défaut utilise le fichier sub.txt :


py txt2sub.py 
py txt2sub.py fichier.txt -o sortie.srt

sous linux:
modifier la 1ere ligne=le chemin de l'executable python
